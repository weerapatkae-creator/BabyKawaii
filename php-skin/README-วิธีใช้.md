# 🌸 BabyKawaii — ดีไซน์ใหม่ทั้งระบบ (ไฟล์ PHP ครบทุกหน้า)

แพ็กนี้แปลงโฉมเว็บทั้งระบบเป็นดีไซน์มินิมอลโทนโรสพาสเทล (เหมือน prototype)
โดย **คงคิวรีฐานข้อมูล ตัวแปร ลอจิก และ ID ที่ผูกกับ JS เดิมทุกตัว** — เปลี่ยนแค่หน้าตา/สี

---

## 📂 โครงสร้างไฟล์ในแพ็ก (วางทับตามตำแหน่งเดิม)

```
assets/css/style.css      → assets/css/style.css      (สกินหลัก คุมทั้งระบบ)
includes/header.php       → includes/header.php       (topbar ใหม่ + search + avatar)
includes/sidebar.php      → includes/sidebar.php      (เมนูจัดกลุ่มแบบ prototype)
dashboard.php             → dashboard.php             (หน้าแรก ดีไซน์เต็ม + แก้บั๊กต้นทุน&กำไร)
login.php                 → login.php
pages/                    → pages/                    (ทั้ง 22 หน้า ปรับสีแบรนด์เป็นโทนใหม่)
```

> **ไม่ต้องแก้ `includes/footer.php`** (ใช้ของเดิม) และ **ไม่ต้องแตะ** `config/`, `api/`, `database/`

---

## ✅ วิธีติดตั้ง

1. **สำรองโปรเจคเดิมทั้งโฟลเดอร์ก่อน** (สำคัญ!)
2. ก๊อปไฟล์/โฟลเดอร์ในแพ็กนี้ไปวางทับตามตำแหน่งด้านบน
3. รีเฟรช (Ctrl+F5 ล้างแคช) ✨

---

## 📌 อะไรเปลี่ยนบ้าง
1. **`style.css`** — รีสกินทุก component (การ์ด/ตาราง/ปุ่ม/แบดจ์/ฟอร์ม/กราฟ) เป็นโทนโรสพาสเทล + ฟอนต์ Noto Sans Thai
2. **`header.php` + `sidebar.php`** — topbar มี search + avatar วงกลม, เมนูจัดกลุ่ม (ภาพรวม / คลังสินค้า / การตลาด & สื่อ / ลูกค้า & ช่องทาง / จัดการระบบ)
3. **`dashboard.php`** — การ์ด "ต้นทุน & กำไร" โทนสว่าง + แก้บั๊กตัวอักษรล้น + กราฟสีโรส
4. **ทุกเพจใน `pages/`** — แทนที่สีแบรนด์ฮาร์ดโค้ดเก่า (ชมพู #FF85A2 / ม่วง #9B72CF ฯลฯ) เป็นโทนใหม่ ให้กลมกลืนทั้งระบบ — **คิวรี/ลอจิก/โครงสร้างเหมือนเดิมเป๊ะ**

## ⚠️ หมายเหตุ
- แถบเมนูข้างคงโทนเข้มไว้ เพราะโลโก้ `logo-light.svg` เป็นสีขาว
- ID ที่ผูกกับ JS (`headerInboxBadge`, `sidebarInboxBadge`, `sidebarToggle` ฯลฯ) คงไว้ครบ — badge แจ้งเตือน/ปุ่มต่างๆ ยังทำงานปกติ
- ฟีเจอร์โต้ตอบเฉพาะใน prototype (เช่น แถบเครื่องมือในแชต, คลังคำตอบอัจฉริยะที่เรียนรู้การปิดการขาย) เป็นฟีเจอร์ใหม่ที่ต้องเพิ่ม **ตาราง DB + โค้ดหลังบ้าน** — แจ้งได้ถ้าต้องการให้ทำส่วนนี้แยก
