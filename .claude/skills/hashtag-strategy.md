# Hashtag Strategy

## When to Use This Skill

- User wants to improve their social media discoverability through hashtags
- User needs organized hashtag sets for different content types
- User is launching a branded hashtag and needs a strategy around it
- User wants platform-specific hashtag guidance (Instagram, TikTok, LinkedIn)
- User asks how many hashtags to use or which ones perform best
- User needs hashtag sets organized by content pillar or topic

## Core Principle

HASHTAGS ARE A DISTRIBUTION STRATEGY, NOT A DECORATION — EVERY HASHTAG MUST CONNECT YOUR CONTENT TO AN AUDIENCE ACTIVELY SEARCHING FOR IT.

## Platform Rules Reference

| Platform | Max Hashtags | Recommended | Placement | Notes |
|----------|-------------|-------------|-----------|-------|
| **Instagram** | 30 | 5-15 | Caption or first comment | Mix of reach tiers. Reels use fewer (3-5). |
| **TikTok** | Unlimited (caption limit) | 3-5 | Caption only | Short, trending, niche. Fewer is better. |
| **LinkedIn** | Unlimited | 3-5 | End of post | Professional and industry-specific only. |
| **X/Twitter** | Unlimited | 1-2 | Inline in tweet | More than 2 reduces engagement. |
| **YouTube** | 60 (description) | 5-8 | Description + up to 3 above title | Use exact search terms viewers type. |
| **Pinterest** | 20 | 2-5 | Pin description | Use as keywords, not community tags. |

## Reach Tier System

| Tier | Post Volume | Your Follower Range | Purpose |
|------|-------------|-------------------|---------|
| **Mega** | 1M+ posts | 50K+ followers | Brand association, aspirational |
| **Broad** | 500K-1M posts | 10K-50K followers | Moderate competition, wider reach |
| **Mid** | 100K-500K posts | 2K-10K followers | Sweet spot for growing accounts |
| **Niche** | 10K-100K posts | Under 2K followers | High relevance, low competition |
| **Micro-niche** | Under 10K posts | Any size | Hyper-targeted, community-specific |
| **Branded** | You create it | Any size | Your own community, UGC collection |

## Recommended Tier Mix per Post

| Account Size | Mega | Broad | Mid | Niche | Micro-niche | Branded |
|-------------|------|-------|-----|-------|-------------|---------|
| Under 1K followers | 0 | 1 | 3 | 4 | 2 | 1 |
| 1K-10K followers | 0 | 2 | 4 | 3 | 1 | 1 |
| 10K-50K followers | 1 | 3 | 4 | 2 | 0 | 1 |
| 50K+ followers | 2 | 3 | 3 | 1 | 0 | 1 |

## Workflow

### Phase 1: Define Content Pillars

1. Identify the user's 3-5 content pillars:
   - What topics do they consistently post about?
   - What does their audience follow them for?
   - What categories align with their products/services?

2. For each pillar, define:
   - The core topic
   - The audience searching for this content
   - Related subtopics and angles

### Phase 2: Research Hashtags by Pillar

3. For each content pillar, build a hashtag bank:
   - Research hashtags across all reach tiers
   - Verify relevance — the hashtag's content must match what the user posts
   - Check for banned or spammy hashtags (Instagram regularly bans common tags)
   - Include a mix of descriptive, community, and action-based hashtags

4. Organize into a hashtag bank table:
   - Columns: Hashtag, Tier, Post Volume (approximate), Pillar
   - Rows: 8-12 hashtags per pillar
   - Total bank: 30-50 hashtags across all pillars

### Phase 3: Create Ready-to-Use Sets

5. Build 3-4 pre-made hashtag sets per pillar:
   - Each set uses the recommended tier mix for the user's account size
   - Each set is different enough to avoid "hashtag repetition penalties"
   - Sets are copy-paste ready (formatted as a single block)

6. Create special-purpose sets:
   - **Launch set:** for product/service announcements
   - **Engagement set:** for community-building posts (questions, polls)
   - **Trending set:** placeholder for current trends (updated monthly)

### Phase 4: Define the Branded Hashtag

7. If the user does not have a branded hashtag, create one:
   - Must be unique (not already in heavy use)
   - Short, memorable, spellable
   - Include the brand name or a signature phrase
   - Test it: search for it to ensure no unwanted associations

### Phase 5: Deliver the Strategy

8. Format the final output as a reference document with:
   - Platform rules summary
   - Full hashtag bank organized by pillar and tier
   - Ready-to-use sets (copy-paste blocks)
   - Branded hashtag with usage guidelines
   - Rotation schedule guidance

## Rotation Schedule

- Rotate between Set A and Set B for each pillar — never use the same set on consecutive posts
- Refresh the hashtag bank monthly: check for banned tags, replace underperformers with new discoveries
- Track which sets drive the most reach in Instagram Insights under "Impressions from Hashtags"

## Recovery and Fallback

- If the user does not know their content pillars, ask: "What are the 3 topics you post about most?" and derive pillars from their answer
- If the user posts on a platform not covered in the reference table, apply the Instagram framework as a baseline and adjust hashtag count to match the platform's norms
- If a recommended hashtag turns out to be banned or spammy, remove it immediately and replace with the next closest niche-tier alternative
- If the user has zero followers (brand new account), focus entirely on niche and micro-niche hashtags — broad tags will bury their content
- After 3 failed attempts to identify relevant hashtags for a niche, stop and say: "Your niche may be too specific for hashtag-driven discovery. Let's focus on content quality and engagement strategy instead of hashtags."

## Constraints

- **NEVER** recommend banned or flagged hashtags — check before including
- **NEVER** recommend more than 15 hashtags for Instagram or more than 5 for TikTok
- **NEVER** use the same hashtag set on every post — rotation prevents algorithm penalties
- Every hashtag must be relevant to the content — generic tags like #love or #instagood waste space
- Always include at least one branded hashtag in every set
- Hashtag banks must cover all content pillars, not just one
- Approximate post volumes must be included so users understand competition level
- All sets must be formatted as copy-paste ready blocks (no bullet points, just the hashtag string)
- Do not include hashtags with special characters or spaces — only valid hashtag format
